Data Deletion Policy

Last updated: July 21, 2026

Local-Only Storage: Because ABX does not collect, upload, or store any personal data on remote servers, there is no remote data to request deletion for. You are in full control of all application data on your local device.

1. User Data Ownership & Local Control

The ABX application operates entirely under a local-only data model. We do not maintain user databases, cloud backups, or remote storage infrastructures. Any diagnostic configurations, log trails, or local preferences exist solely within your device's private sandbox storage.

2. How to Delete All Local Data

You can instantly and completely delete all settings, databases, and local storage cached by the Application at any time:

  1. Open the Settings app on your Android device.
  2. Navigate to Apps or Application Manager.
  3. Select ABX from the installed applications list.
  4. Tap on Storage & Cache.
  5. Tap Clear Storage / Clear Data to purge all local private databases and settings.
  6. Alternatively, Uninstall the application from your device to automatically wipe all sandboxed data directories.

3. No Remote Accounts or Files

Because there are no user accounts or cloud servers linked to ABX, you do not need to request remote account deletion or data extraction from us.
