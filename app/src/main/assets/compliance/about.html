Features & Specifications

This utility enables real-time cryptographic attestation, hardware-backed key pair generation, secure local session management, and on-device sandbox inspection. ABX is designed to run securely and fully offline on your device, ensuring maximum credential isolation inside the Hardware Security Module (HSM) / Secure Enclave.

Architecture

By leveraging Android's robust Keystore system, ABX isolates private key material using non-exportable EC-256 curves. Attestation log integrity is continuously validated and chained with hardware-backed signatures to block untrusted access attempts dynamically.
